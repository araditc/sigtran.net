# References

The SDK is developed against standards-track specifications first, with vendor and implementation references used for interoperability notes and operational behavior.

## Standards

- RFC 4666 - M3UA
- RFC 9260 - SCTP
- RFC 4165 - M2PA baseline

## Supplied Project References

- Tekelec EAGLE SS7-over-IP using SIGTRAN, rev B
- ETSI EG 202 360 V1.1.1 SIGTRAN scenarios
- Nokia MCAS 6.1.1 SS7/SIGTRAN User Guide
- M2PA Internet-Draft 13, treated as historical context where RFC 4165 is the standards-track baseline

## Implementation References

- OpenSS7 SIGTRAN Stack manual pages: http://www.openss7.org/ipss7_man.html
- OpenSS7 source package: https://github.com/openss7/openss7

OpenSS7 references are useful for implementation comparison and terminology, but they do not override RFC behavior in this SDK.
